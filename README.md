# Retail Intelligence Platform

This project is an end-to-end retail analytics portfolio project focused on inventory, sales, transfers, purchase orders, and forecasting.

## Business Objective

The goal is to help retail teams monitor stock availability, sales performance, inventory movement, and forecast accuracy across stores, products, and brands.

## Tools Used

- SQL
- Python
- Power BI
- Excel / Power Query
- GitHub

## Main Modules

1. Sales Analysis
2. Inventory Analysis
3. Purchase Order Tracking
4. Transfer Order Tracking
5. Forecast Accuracy
6. Dashboard Reporting
